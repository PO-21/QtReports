QtReports version 1.0 07/04/2016

-------------------------------------------
Данная разработка написана на языке  программирования C++ с использованием библиотек Qt 
и является подключаемой библиотекой для создания отчетов.

Она позволяет использовать:
-разные источники информации
-геометрические фигуры
-вывод отчета в разных форматах
-работа с изображениями
-кроссплатформенность (проекттестировался на системах: Window 7(64-32),Window 8.1(64-32),Window 10(64),Linux Ubuntu().

-------------------------------------------
Наши контакты:

Сайт проекта :https://github.com/PO-21/QtReports

-------------------------------------------
Требования:

Собрать библиотеки можно на операционных системах Linux и Windows.
Для успешной сборки проекта необходим библиотеки Qt>=5.x и GCC >=4.8
Для Windows необходим утановленный CMake>=3.5.х.
